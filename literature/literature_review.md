# Literature Review and Theoretical Foundations

This document organizes key literature and theoretical concepts relevant to the Hamiltonian of Mean Force (HMF) project. It integrates definitions, exact localization criteria, and an annotated bibliography linked to the BibTeX file `references.bib`.

## 1. Hamiltonian of Mean Force (HMF) Programme

### Definitions and the Reduced Equilibrium Object
The total Hamiltonian is given by $H_{\mathrm{tot}} = H_Q + H_B + H_I$, where $H_Q$ acts on the system Hilbert space, $H_B$ on the bath, and $H_I$ couples them.

The reduced (unnormalised) equilibrium operator at inverse temperature $\beta$ is:
$$ \rho_S^{\mathrm{unnorm}}(\beta)=\mathrm{Tr}_B e^{-\beta H_{\mathrm{tot}}} $$
The normalised reduced state is $\rho_S(\beta)=\rho_S^{\mathrm{unnorm}}/\mathrm{Tr}_S\rho_S^{\mathrm{unnorm}}$. This is the starting point for the HMF programme.

A standard operator definition of the quantum HMF factors out the bare-bath partition function $Z_B(\beta)=\mathrm{Tr}_B e^{-\beta H_B}$ and defines the mean-force Gibbs object:
$$ e^{-\beta H_{\mathrm{MF}}(\beta)} \propto \frac{\mathrm{Tr}_B e^{-\beta(H_Q+H_B+H_I)}}{\mathrm{Tr}_B e^{-\beta H_B}} \quad\Longrightarrow\quad H_{\mathrm{MF}}(\beta)= -\beta^{-1}\ln\left(\frac{\mathrm{Tr}_B e^{-\beta(H_Q+H_B+H_I)}}{Z_B(\beta)}\right) $$
(up to an additive scalar).

**Key Points:**
1.  **Exactness:** $H_{\mathrm{MF}}(\beta)$ is an exact restatement of the reduced equilibrium state. No approximation enters until one attempts to represent it in a restricted operator class (e.g., "local Hamiltonians").
2.  **Correlations:** Equilibrium correlations are non-optional at strong coupling. The reduced equilibrium operator is generated from a globally thermal state. Methods initializing in the coupled equilibrium state target the reduced object $\mathrm{Tr}_B e^{-\beta H_{\mathrm{tot}}}$ directly.

## 2. Exact Imaginary-Time Influence Functional

### Imaginary-Time Interaction Representation
The core identity relies on the decomposition $H_{\mathrm{tot}} = H_0 + V$:
$$ e^{-\beta(H_0+V)} = e^{-\beta H_0} \mathcal{T}_\tau \exp\left(-\int_0^\beta d\tau V_I(\tau)\right), \qquad V_I(\tau)=e^{\tau H_0} V e^{-\tau H_0} $$
where $\mathcal{T}_\tau$ is imaginary-time ordering.

For $H_I = f \otimes B$, the interaction-picture operator is $\tilde f(\tau) \equiv e^{\tau H_Q} f e^{-\tau H_Q}$. Note that $\tilde f(\tau)$ is time-dependent in the interaction picture even if $f$ is Schrödinger-picture time-independent.

### Bilocal Euclidean Influence Functional
For a harmonic bath linearly coupled to a system operator, tracing out the bath yields an exact bilocal functional. The result depends on a kernel built from bath correlators.

**Gaussian-Field Reformulation:**
The Hubbard-Stratonovich (HS) linearisation converts quadratic forms into Gaussian averages over auxiliary fields.
This connects the bilocal quadratic influence functional to an average over a (generally coloured) Gaussian process coupled linearly to system operators.
-   **Method:** Euclidean influence functional $\to$ Stochastic/Gaussian-field representation.
-   **Relation to Partition-Free:** This aligns with partition-free dynamics which formulate open-system dynamics starting from coupled equilibrium, using influence functionals and HS transformations.

## 3. Operator-Level Localisation: Closure Criterion

### Adjoint-Action Expansion
The interaction-picture coupling admits the exact adjoint-action expansion:
$$ \tilde f(\tau) = e^{\tau H_Q} f e^{-\tau H_Q} = e^{\tau \mathrm{ad}_{H_Q}}(f) = \sum_{n=0}^\infty \frac{\tau^n}{n!} \mathrm{ad}_{H_Q}^n(f), \qquad \mathrm{ad}_{H_Q}(X) \equiv [H_Q, X]. $$

### Closure Criterion for Exact Localisation
A criterion for representing $H_{\mathrm{MF}}(\beta)$ within a target operator class $\mathcal{A}_{\mathrm{loc}}$:
An exact representation is possible only if the operator algebra generated by the repeated adjoint actions of $H_Q$ on the coupling operators closes within $\mathcal{A}_{\mathrm{loc}}$.

If $\{f, \mathrm{ad}_{H_Q}(f), \mathrm{ad}_{H_Q}^2(f), \dots\}$ generates operators outside the ansatz, the Magnus/BCH expansions imply that $\ln(\dots)$—and thus $H_{\mathrm{MF}}$—will contain those non-local operators. This provides a rigorous condition for when a local Hamiltonian representation is exact vs. approximative.

### Solvable Closures
-   **Commuting Coupling ($[H_Q, f]=0$):** Adjoint action vanishes for $n \ge 1$. $\tilde f(\tau) = f$. Exact closure.
-   **Quadratic/Gaussian Systems:** Total state is Gaussian, reduced state is Gaussian, $H_{\mathrm{MF}}$ is quadratic. Closure within quadratic operators.
-   **Single Qubit (Pauli Algebra):** Algebra closes on $\mathfrak{su}(2)$. Finite operator basis allows closure satisfaction in a non-trivial way.

## 4. Annotated Bibliography

### Exact Influence Functional & Gaussian Reformulation

-   **feynmanVernon1963Influence**
    *Foundational influence functional derivation for linear dissipative environments.* Template for integrating out a harmonic bath.
-   **caldeiraLeggett1983Tunnelling**
    *Canonical dissipative-system model.* Standard reference for Gaussian bath class and path integrals.
-   **grabertSchrammIngold1988QBM**
    *Comprehensive functional-integral review.* Covers equilibrium correlation functions and influence functionals.
-   **hubbard1959HS**
    *Original HS partition-function manipulation.* Exact Gaussian auxiliary-field linearisation.
-   **stratonovich1957QDistro**
    *Early HS method statement.* Historical source for Gaussian-field linearisation.
-   **mccaulLorenzKantorovich2017PartitionFree**
    *Partition-free approach.* Influence functional + two-time HS transformation; relevant to Gaussian field reformulation.
-   **moixZhaoCao2012EquilibriumRDM**
    *Stochastic unravelling of imaginary-time influence functional.* Constructing equilibrium RDM.
-   **chenXin2014StochasticMatrix**
    *Rigorous stochastic matrix multiplication.* Interpreting Euclidean influence functional as Gaussian-process average.
-   **songShi2015CorrelatedInitialHEOM** & **tanimura2014RealImagHEOM**
    *Imaginary-time path integrals in HEOM.* Correlated initial states and thermodynamic quantities.
-   **mccaul2020ControllingObservables**
    *Controlling arbitrary observables in correlated many-body systems.* Relevant to partition-free themes.
-   **mccaul2025ObservablesInMotion**
    *Observables in Motion.* Comprehensive guide to simulating classical and quantum dynamics using influence functionals.
