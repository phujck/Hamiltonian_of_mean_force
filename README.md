# Hamiltonian of Mean Force

## Overview
Research project investigating the Hamiltonian of Mean Force.

## Directory Structure
- `theory/`: Theoretical derivations and central calculations.
- `simulations/`: Python simulation code.
- `manuscript/`: Physical Review style manuscript (LaTeX).
- `literature/`: Literature review notes and references.

## Usage
Use the `manage.ps1` script to manage the project:

```powershell
./manage.ps1 install   # Install dependencies
./manage.ps1 sim       # Run simulations
./manage.ps1 paper     # Compile manuscript
./manage.ps1 clean     # Clean build artifacts
```
